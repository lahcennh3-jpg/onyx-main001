## Redaction Requirements

Redact before publishing:

- Tenant URLs
- User names and emails
- Account IDs
- API keys and key prefixes
- Connector names that expose private systems
- Internal URLs
- Document names that reveal sensitive details
- Raw prompt or transcript content

## Capture Rule

Publish redacted screenshots only after review and secret scan. If redaction quality is uncertain, publish a written redacted summary instead.
